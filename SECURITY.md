# Security Policy

## Supported Versions

Only the latest version on the `master` branch is supported with security fixes.

| Version | Supported |
| ------- | --------- |
| Latest (`master`) | :white_check_mark: |
| Older versions / forks | :x: |

## Reporting a Vulnerability

If you discover a security issue, please report it privately by email:

- **Email:** alinoposta@gmail.com
- **Subject:** `[GEOP Importer][Security] <short title>`

Please include:
- clear description of the issue
- reproduction steps
- affected file/section
- potential impact
- optional fix suggestion

### Response timeline
- Initial response: within **72 hours**
- Status update: within **7 days**
- Fix timeline: as soon as possible based on severity

Please do **not** open public GitHub issues for security vulnerabilities.
